    Introduction

    Welcome to the Kittn API! You can use our API to access Kittn API endpoints, which can get information on various cats, kittens, and breeds in our database.

    We have language bindings in Shell, Ruby, Python, and JavaScript! You can view code examples in the dark area to the right, and you can switch the programming language of the examples with the tabs in the top right.

    This example API documentation page was created with Slate. Feel free to edit it and use it as a base for your own API's documentation.

    This endpoint retrieves all screen.

    HTTP Request

    GET https://signage.pickcel.com/api/v4/client/screens/list?page=0&limit=2&sort_by=first_name&order=asc

    Query Parameters

    Parameter Default Description
    page 0 page number, start with 0 being first page
    limit N.A. number of results returned per page, value should be greater than 1 and less than 30
    sort_by N.A. defines the field which needs to be sorted by
    order dsc defines how sorting has to be done, values can be either asc [ascending] or dsc [descending]. If not passed descending will be default
    screen_status all online, offline, deactivated, when the field is not passed, the results would have all data without any filter

    This endpoint retrieves all media.

    HTTP Request

    GET https://signage.pickcel.com/api/v4/client/media/list?asset_type=all&page=3&limit=10&sort_by=modified_date&order=asc

    Query Parameters

    Parameter Default Description
    page 0 page number, start with 0 being first page
    limit N.A. number of results returned per page, value should be greater than 1 and less than 30
    sort_by N.A. defines the field which needs to be sorted by
    order dsc defines how sorting has to be done, values can be either asc [ascending] or dsc [descending]. If not passed descending will be default
    asset_type all asset type - image/video/pdf/apps
